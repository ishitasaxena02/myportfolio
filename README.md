# Ishita Saxena | Portfolio

A developer portfolio built with **HTML5**, **CSS3**, and **JavaScript** — featuring a cyberpunk-inspired terminal UI, interactive chatbot, and section-based animations.

<a href="https://ishitasaxena02.github.io/myportfolio/" target="_blank">**Visit Now**</a>

## Sections

- **Hero** — Terminal-style intro with typing animation
- **About** — Interactive chatbot conversation
- **Skills** — VS Code-themed IDE with categorized tech stack
- **Experience** — Timeline with animated walking figure
- **Education** — Classroom projector theme with performance stats
- **Projects** — GitHub repository cards
- **Contact** — Terminal form with mailto integration

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (no frameworks)

## Run Locally

```bash
git clone https://github.com/ishitasaxena02/myportfolio.git
cd myportfolio
```

Open `index.html` in your browser.
